## Проект 3 — Python-анализ трафика и регистраций образовательной платформы
Стек: Python (pandas, matplotlib, seaborn, plotly, requests, python-dotenv), Jupyter Notebook, rest api, GitHub

🛠 Описание:
1. Запрашивал данные по визитам и регистрациям через REST API (параметризованные GET-запросы), настроил конфигурацию через переменные окружения (.env + python-dotenv) для переносимости скрипта
2. Выполнил очистку данных: отфильтровал SEO/поисковых ботов по User-Agent, дедуплицировал визиты по visit_id (оставил последний визит перед регистрацией), рассчитал конверсию из визита в регистрацию в разрезе платформ (web, ios, android)
3. Объединил данные конверсий с рекламными кампаниями из CSV, рассчитал стоимость рекламы по датам и кампаниям
4. Построил 9 графиков (PNG): суммарные визиты и регистрации, разбивка по платформам, конверсия по платформам, средняя конверсия, стоимость рекламы, визиты и регистрации с цветовым выделением рекламных кампаний — с использованием matplotlib, seaborn, plotly
5. Подготовил Jupyter Notebook (charts_project.ipynb) с полным воспроизводимым пайплайном: загрузка → обработка → расчёт метрик → визуализация; все артефакты сохранены в структурированные директории
